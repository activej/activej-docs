基准

我们使用JMH作为基准工具测量了 HTTP GET 的请求性能。

时间：29112ms；平均时间：5822.4ms；最佳时间：5757ms；最差时间：5892ms；每秒请求：171750。

ActiveJ 结果是 171.8K RPS 在单个CPU上。 它不仅比高度专业化的 Vert.x （162.9K RPS），而且CPU负载减少50%。

当使用HTTP管道时，性能达到 1.5M RPS。

根据 2021-08-31，ActiveJ在 [the-benchmarker的网络框架基准测试](https://github. com/the-benchmarker/web-frameworks) ，排名第一。

至于 TechEmpower基准的连续运行，ActiveJ HTTP在 JSON 和 Plaintext 类别中名列顶级Java服务器。
