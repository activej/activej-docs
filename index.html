ActiveJ Components

Inject

Lightning-fast and ultimately powerful DI library

Dependency injection library with ultimate performance and no third-party dependencies. Multi-thread friendly, feature-rich, and can boast with lighting-fast start-up time and runtime.

Serializer

The fastest JVM serializer in the world

Fastest Java serializer in the world. At the same time, it is extremely powerful and features full support of Java subclasses, collections (including Maps), and also specialized collections like HPPC.

Codegen

Generation without overhead of reflection

ActiveJ Codegen is an efficient Java code generator with streamlined and simple API. It is built on top of ObjectWeb ASM and has zero-overhead performance. ActiveJ Codegen is widely utilised at ActiveJ Serializer

Specializer

A little bit of magic to speed up your code

ActiveJ Specializer is an innovative technology for enhancing class runtime performance for JVM. It is one of the ActiveJ technologies with no third-party dependencies, so it can be used as a stand-alone component.

FS

Managing local, remote, and distributed FS

ActiveJ FS is a lightweight asynchronous Java library that provides a tiny abstraction on top of common file operations. It provides upload, download, append, list, copy, move, delete, and other methods for operating with storage.

RPC

Binary protocol for microservices architecture

Lightweight and convenient Java library for developing high-load distributed applications. Completely alternative approach to microservices implementation and overcomes the overheads of utilizing HTTP protocol.
