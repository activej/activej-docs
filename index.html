Java framework for modern web, cloud, high-load, and microservices solutions

Components

Inspired by Node.js approach and has almost no third-party dependencies

Async-IO

Asynchronous IO processing

Contain event loop, promises, streaming protocols, asynchronous sockets and a TCP server with TLS support

HTTP

Asynchronous server and client

Support streaming, pipelining and WebSocket. Features simple yet flexible API. Suitable for processing millions of requests per second

Inject

Dependency injection library

DI with ultimate performance and no third-party dependencies. Multi-thread friendly, feature-rich, fast start-up time and runtime

Serializer

The fastest JVM serializer

Features full support of Java subclasses, collections (including Maps), and also specialized collections like HPPC

Codegen

No reflection overhead

Code generator with simple API. It is built on top of ObjectWeb ASM and has zero-overhead performance. Widely utilised at ActiveJ Serializer

Specializer

Magic to speed up your code

Technology for enhancing class runtime performance for JVM. No third-party dependencies, can be used as a stand-alone component

FS

Managing file systems

Asynchronous Java library that is a tiny abstraction on top of common file operations. Provides methods for operating on a file system

RPC

Binary protocol for microservices

Library for high-load distributed applications. An alternative to microservice architecture with overhead of HTTP
